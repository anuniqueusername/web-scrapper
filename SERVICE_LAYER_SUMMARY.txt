✅ TROUBLESHOOTING:

Can't start scraper?
  → Check scraper.js exists
  → Check Node.js is installed
  → Review scraper.log for errors

Can't stop scraper?
  → Wait 5 seconds (graceful shutdown timeout)
  → Restart via Restart button
  → Or manually: rm scraper.pid

Dashboard shows PID but nothing happening?
  → Check scraper.log for errors
  → Verify configuration is enabled
  → Try clicking Restart

Port 3000 in use?
  → Kill other process on port 3000
  → Or use: PORT=3001 npm start
